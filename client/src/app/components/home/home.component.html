Kochii

Introduction

Kochii is a real-time inventory management web application intended to keep track and organize ingredients. It allows the user to categorize each item by group as well as receive warnings regarding the status of each item whether expired, almost expired, or good. The actions of adding and removing items are tracked by the user's history component which can be cleared whenever the user wants. The history is used to populate the overview component with data to summarize the user's weekly activities to provide useful insights.

About

Kochii is a personal project created by Jeffrey Ram Pineda as a hobby. It is created with Angular as a frontend framework, Koa.js for the backend server, and MongoDB as its database. Still an ongoing hobby, Kochii is updated from time to time with no specific schedules in mind. The project is open-sourced and is hosted on github.com/jeffreyrampineda.
